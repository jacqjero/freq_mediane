Description:
Fonction fmediane qui permet à partir d'un fichier de données Input.txt de calculer les fréquences limites et centrale d'un filtre nieme d'octave

Ce projet contient 3 fichiers:

- Input.txt: fichier contenant les inputs pour la routine fmediane.py
- fmediane.py: routine qui calcul les fréquences limites et centrale des bandes nième d'octave
- recomposition_spectrale.py: fonction appelée par fmediane
